== Description ==

My Extreme Facebook plugin will add an widget to use facebook fan page like box.

Plugin Features

* Fan page stream
* Header enable/Disable
* Theme selection
* You have many option how you want to use your fan page like page.
* Very Lightweight.
* Easy to use
& many More.

== Installation ==

* Upload `myextremefacebook.zip` to the `/wp-content/plugins/` directory and extract it.
* Activate the plugin through the 'Plugins' menu in WordPress
* Use widget "My Extreme Facebook" and Put your fan page URL.
* Now customize it Like select theme, Use header, Stream and so forth.

== Frequently asked questions ==

= Will it slow down my website =
My answer is no. It is very light weight. Using Iframe. No script.

= My fanpage stream and faces are not loading =
May be you forgot to add your fanpage url. Go to widget section and put your fanpage url in Facebook page URL section.
